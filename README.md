#  BMI Calculator App
 The BMI Calculator is a simple Android application built using Kotlin that allows users to calculate their Body Mass Index (BMI). Based on the BMI value, users can get a quick insight into their weight statusâ€”whether they are underweight, normal weight, overweight, or obese. 
## Features
User-friendly interface for inputting weight and height.
Option to input weight in kilograms and height in centimeters or meters.
Displays calculated BMI result.
Provides a categorized result (Underweight, Normal weight, Overweight, Obese).
Clear and reset functionality.
###  Used libraries
DataBinding  , os.Bundle.
